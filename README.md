# 🚛 TruckLog Pro - Virtual Trucking Companion

แอปพลิเคชันจัดการบัญชีและบันทึกข้อมูลการเดินทางสำหรับผู้เล่นเกม **Euro Truck Simulator 2 (ETS2)** และ **American Truck Simulator (ATS)** โดยเฉพาะ ช่วยให้คุณบริหารจัดการรายได้ รายจ่าย และประสิทธิภาพการขับรถได้อย่างมืออาชีพ

---

## ✨ คุณสมบัติเด่น
- 📊 **Dashboard:** สรุปภาพรวมรายได้ กำไร และสถิติการขับขี่
- 🛣️ **Trip Logger:** บันทึกรายละเอียดการเดินทาง ระยะทาง ค่าน้ำมัน และค่าปรับ
- 💰 **Accounting:** ระบบบัญชีแยกประเภท (Ledger) จัดการรายรับ-รายจ่าย
- 🚛 **Fleet Management:** ติดตามข้อมูลรถบรรทุกและพนักงานขับรถในสังกัด
- ☁️ **Hybrid Backend:** รองรับการทำงานทั้งแบบ Standalone (Python) และบน Web Server (XAMPP/PHP)
- 💾 **Auto Sync:** บันทึกข้อมูลลงไฟล์ `database.json` โดยอัตโนมัติ

---

## 🚀 วิธีการติดตั้งและใช้งาน

### วิธีที่ 1: ใช้งานผ่าน Python (แนะนำ - ง่ายที่สุด)
เหมาะสำหรับการใช้งานเครื่องเดียว ไม่ต้องตั้งค่าเซิร์ฟเวอร์ซับซ้อน

1. **ติดตั้ง Python:** ดาวน์โหลดและติดตั้งจาก [python.org](https://www.python.org/)
2. **เริ่มใช้งาน:**
   - ไปที่โฟลเดอร์ของโปรเจกต์
   - ดับเบิลคลิกไฟล์ `run_trucklog.bat`
   - (หรือพิมพ์คำสั่ง `python server.py` ใน Terminal)
3. **เปิดเบราว์เซอร์:** เข้าไปที่ `http://localhost:3000`

---

### วิธีที่ 2: ใช้งานผ่าน XAMPP (สำหรับสาย Web Server)
เหมาะสำหรับผู้ที่มี XAMPP อยู่แล้ว หรือต้องการเปิดให้เครื่องอื่นในวงแลนเข้าใช้งานได้

1. **คัดลอกไฟล์:** ก๊อปปี้โฟลเดอร์โปรเจกต์ทั้งหมดไปวางที่ `C:\xampp\htdocs\VTC`
2. **เปิด Apache:** เปิด **XAMPP Control Panel** และกด **Start** ที่ Apache
3. **เปิดเบราว์เซอร์:** เข้าไปที่ `http://localhost/VTC/index.html`

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- **Frontend:** HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Backend:** 
  - Python (http.server) - สำหรับการใช้งานแบบ Standalone น้ำหนักเบา
  - PHP - สำหรับใช้งานร่วมกับ Apache/XAMPP
- **Database:** เก็บข้อมูลในรูปแบบไฟล์ JSON (`database.json`)

---

## ⚠️ การสำรองข้อมูล (Data Backup)
ข้อมูลทั้งหมดของคุณจะถูกเก็บไว้ในไฟล์ชื่อ **`database.json`** ในโฟลเดอร์หลักของโปรเจกต์ หากต้องการสำรองข้อมูลหรือย้ายเครื่อง เพียงแค่คัดลอกไฟล์นี้ไปวางในที่ใหม่ได้เลย
